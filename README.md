## 🛡️ Executive Summary

I don't build features; I build **robust infrastructures**. I specialize in creating backend systems where **security, separation of concerns, and controlled failure** are the primary requirements. I believe that if a system isn't "boring" to maintain, it wasn't designed correctly.

* **Defensive Design:** Implementing RBAC, JWT, and OAuth2 with a zero-trust mindset.
* **Architectural Rigor:** Heavy focus on **Domain-Driven Design (DDD)** and **Clean Architecture**.
* **Automation:** CI/CD pipelines that enforce security through SAST/DAST scanning.
