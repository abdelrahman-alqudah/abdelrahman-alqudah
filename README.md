## 🔴 About Me

I'm a **Back-End Developer** focused on **Security, Architecture, and Reliability**.
I build systems the way financial institutions expect them:
**predictable, hardened, and boringly stable**.

I don't chase hype — I chase **correctness, separation of concerns, and controlled failure**.

### What I actually do

- 🏛️ Design **secure backend systems** with **ASP.NET Core & SQL Server**
- 🔐 Implement **JWT Authentication**, **RBAC**, and security-first APIs
- 🐳 Containerize services with **Docker**
- 🎯 Apply **SOLID principles** and **Clean Architecture** — intentionally, not as buzzwords
